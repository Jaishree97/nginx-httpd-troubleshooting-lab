# Apache HTTPD Concepts

## What is Apache HTTP Server?

Apache HTTP Server is an open-source web server used for hosting websites and web applications.

---

# Default Port

```text
80
```

---

# Custom Port Used in Lab

```text
83
```

---

# Features of Apache HTTPD

- Open-source
- Modular architecture
- Virtual host support
- Secure web hosting
- Dynamic content support

---

# Important Paths

| Purpose | Path |
|---|---|
| Configuration File | /etc/httpd/conf/httpd.conf |
| Website Files | /var/www/html |
| Logs | /var/log/httpd/ |
