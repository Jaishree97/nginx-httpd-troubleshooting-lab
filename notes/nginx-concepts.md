# NGINX Concepts

## What is NGINX?

NGINX is a high-performance web server and reverse proxy server.

It is commonly used for:
- Web hosting
- Reverse proxy
- Load balancing
- Caching

---

# Default Port

```text
80
```

---

# Features of NGINX

- Lightweight
- High performance
- Fast request handling
- Reverse proxy support
- Load balancing support

---

# Important Paths

| Purpose | Path |
|---|---|
| Configuration File | /etc/nginx/nginx.conf |
| Website Files | /usr/share/nginx/html |
| Logs | /var/log/nginx/ |
